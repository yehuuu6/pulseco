# Pulseco

Pulseco is a highly customizable, secure and stable chat server created with Python that supports third party plugins.

## Features

- **Security**: Implements robust encryption to ensure all messages are secure.
- **Stability**: Built to handle high traffic with minimal downtime.
- **Scalability**: Easily scalable to accommodate growing user bases.

## Installation

To install Pulseco, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yehuuu6/pulseco.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pulseco
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To start the Pulseco server, run:

```bash
python server.py
```
